## インタラクション設計

### 地図操作
1. **ドラッグ**: 地図を移動
2. **ピンチ/ホイール**: ズームイン/アウト
3. **ダブルタップ/クリック**: その地点にズーム
4. **マーカークリック**: カフェ情報ポップアップ

### カフェ情報ポップアップ
```
┌─────────────────────┐
│ Blue Bottle Coffee  │
│ ★★★★★              │
│ 渋谷区神宮前...      │
│ [詳細を見る]        │
└─────────────────────┘
```

### トランジション
- **画面遷移**: 左右スライド（0.3秒）
- **カード出現**: フェードイン＋上スライド
- **削除時**: フェードアウト＋縮小

## レスポンシブ対応

### デスクトップ（992px以上）
```
┌────────────┬─────────────────────────┐
│            │                         │
│  サイド     │      地図エリア          │
│  バー       │                         │
│            │                         │
│ ・地図      │                         │
│ ・リスト    │                         │
│ ・新規追加  │                         │
│ ・設定      │                         │
│            │                         │
└────────────┴─────────────────────────┘
```

### タブレット（768px-991px）
- 地図とリストを切り替え表示
- フローティングアクションボタン

### モバイル（767px以下）
- フルスクリーン表示
- ボトムナビゲーション
- スワイプでページ切り替え

## アクセシビリティ配慮

### キーボード操作
- Tab: フォーカス移動
- Enter: 選択/実行
- Esc: ダイアログ閉じる
- 矢印キー: 地図移動

### スクリーンリーダー対応
- 適切なARIAラベル
- ランドマーク設定
- フォーカス管理

### タッチ操作
- 最小タップ領域: 44x44px
- スワイプジェスチャー対応
- 長押しメニュー
